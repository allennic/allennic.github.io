redis统计实时在线人数

发表于 2016-11-10   |   分类于 技术
  • SADD
  • SCARD
  • SREM

SADD

我们在密码验证成功页面添加（$user_id是密码验证正确后，从数据库取到redis）
$redis->sadd(“online_users”,$user_id);

SCARD

在个人主页显示在线人数
$redis->scard(“online_users”);

严重参考huangz

SREM

在退出页面添加
$redis->srem(“online_users”,$user_id);

如果不删除会导致在线人数一直呈上升趋势。

后记:

刚开始试用zadd，但是要增加一项时间戳（上线时间），导致不能从有序集合里删除$user_id，所以不能正确统计在线人数。

在nginx配置naxsi

一、命令行复制naxsi_config中的naxsi_core.rules到nginx的etc配置目录

二、在nginx的etc配置目录新建一个naxsi.rules

代码如下

SecRulesEnabled;

DeniedUrl “/RequestDenied”;

CheckRule “$SQL >= 8” BLOCK;

CheckRule “$RFI >= 8” BLOCK;

CheckRule “$TRAVERSAL >= 4” BLOCK;

CheckRule “$EVADE >= 4” BLOCK;

CheckRule “$XSS >= 8” BLOCK;

三、

在http{}中设置include ../naxsi_core.rules;

在server{}中设置 include ../naxsi.rules;
